# Save

```
save [ID]
```

This command saves the playerdata of the specified user.

## Arguments

| Argument | Data Type | Optional | Default Value | Explanation               |
| -------- | --------- | -------- | ------------- | ------------------------- |
| ID       | number    | No       | -             | The ServerID of the user. |

## Note

> This will only save the playerdata of the ID specified.
